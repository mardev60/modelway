Quickstart

To get started with ModelWay, follow these simple steps:

1. Install the SDK

npm install modelway-sdk

2. Initialize the Client

import {{ '{' }} ModelWay {{ '}' }} from 'modelway-sdk';

const modelway = new ModelWay({{ '{' }}
  apiKey: 'your-api-key'
{{ '}' }});

3. Make Your First Request

const response = await modelway.chat.completions.create({{ '{' }}
  model: "gpt-4",
  messages: [
    {{ '{' }} role: "user", content: "Hello!" {{ '}' }}
  ]
{{ '}' }});

Authentication

ModelWay uses API keys for authentication. You can find your API key in the settings page.

Never share your API keys or commit them to version control.

Making Requests

ModelWay automatically routes your requests to the best performing model instance. Here's how to make different types of requests:

Chat Completions

const chatResponse = await modelway.chat.completions.create({{ '{' }}
  model: "gpt-4",
  messages: [
    {{ '{' }} role: "system", content: "You are a helpful assistant." {{ '}' }},
    {{ '{' }} role: "user", content: "What is ModelWay?" {{ '}' }}
  ],
  temperature: 0.7
{{ '}' }});

Streaming Responses

const stream = await modelway.chat.completions.create({{ '{' }}
  model: "gpt-4",
  messages: [{{ '{' }} role: "user", content: "Tell me a story" {{ '}' }}],
  stream: true
{{ '}' }});

for await (const chunk of stream) {{ '{' }}
  console.log(chunk.choices[0]?.delta?.content);
{{ '}' }}

Error Handling

ModelWay provides detailed error messages to help you debug issues:

try {{ '{' }}
  const response = await modelway.chat.completions.create({{ '{' }}
    model: "gpt-4",
    messages: [{{ '{' }} role: "user", content: "Hello!" {{ '}' }}]
  {{ '}' }});
{{ '}' }} catch (error) {{ '{' }}
  if (error.code === 'insufficient_quota') {{ '{' }}
    console.log('Please upgrade your plan');
  {{ '}' }} else if (error.code === 'model_not_available') {{ '{' }}
    console.log('Try with a different model');
  {{ '}' }}
{{ '}' }}
Error Code Description
insufficient_quota Your current plan's quota has been exceeded
model_not_available The requested model is currently unavailable
invalid_request The request parameters are invalid
