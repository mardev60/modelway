Quickstart

ModelWay is designed to work as a drop-in replacement for OpenAI's API. You can continue using your existing OpenAI SDK implementation by simply changing the base URL.

Authentication

ModelWay uses API keys for authentication. You can find your API key in the settings page.

Never share your API keys or commit them to version control.

SDK Integration

You can use the official OpenAI SDK with ModelWay by setting the base URL:

JavaScript/Node.js

import OpenAI from 'openai';

const openai = new OpenAI({{ '{' }}
  apiKey: 'your-modelway-api-key',
  baseURL: 'https://api.modelway.com/v1'
{{ '}' }});

Python

from openai import OpenAI

client = OpenAI(
    api_key="your-modelway-api-key",
    base_url="https://api.modelway.com/v1"
)

API Examples

JavaScript/Node.js

const completion = await openai.chat.completions.create({{ '{' }}
  model: "openai/gpt-4o",
  messages: [
    {{ '{' }} role: "system", content: "You are a helpful assistant." {{ '}' }},
    {{ '{' }} role: "user", content: "Hello!" {{ '}' }}
  ]
{{ '}' }});

Python

completion = client.chat.completions.create(
    model="openai/gpt-4o",
    messages=[
        {{ '{' }}"role": "system", "content": "You are a helpful assistant."{{ '}' }},
        {{ '{' }}"role": "user", "content": "Hello!"{{ '}' }}
    ]
)

cURL

curl https://api.modelway.com/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer your-modelway-api-key" \
  -d '{{ '{' }}
    "model": "openai/gpt-4o",
    "messages": [
      {{ '{' }}"role": "system", "content": "You are a helpful assistant."{{ '}' }},
      {{ '{' }}"role": "user", "content": "Hello!"{{ '}' }}
    ]
  {{ '}' }}'

Error Handling

ModelWay provides detailed error messages to help you debug issues:

try {{ '{' }}
  const response = await openai.chat.completions.create({{ '{' }}
    model: "openai/gpt-4o",
    messages: [{{ '{' }} role: "user", content: "Hello!" {{ '}' }}]
  {{ '}' }});
{{ '}' }} catch (error) {{ '{' }}
  if (error.code === 'insufficient_quota') {{ '{' }}
    console.log('Please upgrade your plan');
  {{ '}' }} else if (error.code === 'model_not_available') {{ '{' }}
    console.log('Try with a different model');
  {{ '}' }}
{{ '}' }}
Error Code Description
insufficient_quota Your current plan's quota has been exceeded
model_not_available The requested model is currently unavailable
invalid_request The request parameters are invalid
